# yugioh_linebot
串接gemini API，下固定指令詞，使其可以掃描卡牌照片快速的查詢卡片效果跟戰術分析
查詢當下最新禁卡表
為玩家在上面管理牌組，可以在有想法時建立牌組構思，支援增刪改查，不會一下就固定無法改變
決鬥時會需要硬幣跟骰子還有計算機的功能，在上面也都準備OK，不用在像過往使用一般計算機可能會有按錯的瑕疵，直接選擇我方與對手就可以計算生命值，即使使用其他功能也不會影響後續計算
串聯line bot可以，不用下載多餘的程式，不用再攜帶硬幣或是骰子更加的方便利
